About This Project

This DonorsChoose.org Project focuses on different poverty types of school projects throughout the Bay Area. Data was fetched utilizing donorschoose.org public source API, being limited to 50 calls maximum per Bay Area city in California. Overall, 102 cities in the Bay Area were analyzed for school projects, household average salary, and the total cost needed to complete all projects.

All analysis and data gathering were all compiled into Jupyter Notebooks using Python 3.6. The main libraries used were matplotlib and seaborn to generate the graphs for this project. The household salary average was derived from https://www.bestplaces.net/, a site dedicated to hosting data for cities across the nation. An ORM database + postgres file has also been generated for future improvements to have it available via Heroku.
